# bigfiximport

Command line tool to assist in importing content into IBM BigFix utilizing Jinja2 XML templates.

## Requirements

* besapi - https://github.com/CLCMacTeam/besapi
* hachoir-metadata
* lxml
* requests
* munkilib (included)
